# Todo-C — 命令行待办与番茄钟

一个用 C 语言编写的命令行待办事项管理工具，集成了任务管理与番茄专注计时功能。这是我学习 C 语言过程中的课程设计作品。

## ✨ 功能特性

- **任务管理**：支持任务的增、删、查
- **优先级系统**：任务分为低、中、高三个优先级
- **专注计时**：内置倒计时番茄钟，记录每次专注时长
- **任务完成追踪**：标记任务完成状态，记录完成时间
- **今日报告**：自动汇总当天完成的任务
- **数据持久化**：任务数据自动保存到本地文件，下次启动自动加载
- **跨平台支持**：同时支持 Windows 和 Linux / macOS

## 📁 项目结构

```
todo-c/
├── todo.c          # 主程序源代码
├── task.txt        # 任务数据文件（程序自动生成）
```

## 🛠 编译与运行

### 编译

使用 GCC 编译：

```bash
gcc todo.c -o todo
```

### 运行

```bash
./todo      # Linux
todo.exe    # Windows
```

## 📖 使用说明

程序启动后进入主菜单，按提示输入数字即可操作：

| 选项 | 功能 |
|------|------|
| `1` | 新建任务 |
| `2` | 删除任务（按 ID） |
| `3` | 查看全部任务 |
| `4` | 查看未完成任务 |
| `5` | 开始专注（番茄钟） |
| `6` | 查看今日报告 |
| `-1` | 保存并退出 |

### 新建任务

依次输入任务名称、任务描述，然后选择优先级（1=低 / 2=中 / 3=高）。

### 开始专注

1. 输入要专注的任务 ID
2. 输入专注时长（分钟）
3. 程序进入倒计时界面，实时显示进度
4. 倒计时结束后，询问任务是否完成

### 数据存储

所有任务数据自动保存在 `task.txt` 文件中，程序启动时自动加载，退出时自动保存。
